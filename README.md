The Steampowered Skyhook
========================

A wrapper for the Steam web API.

Installation
------------

Add this line to your application's Gemfile:

```ruby
gem 'skyhook'
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install skyhook
```
